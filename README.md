Project Overview

Diwali Sales Analysis ek data analysis project hai jisme Python ka use karke Diwali festival ke sales data ko analyze kiya gaya hai. Is project ka main objective customer behavior, purchasing patterns, aur sales trends ko samajhna hai taaki businesses apni marketing strategy improve kar sakein.

🎯 Objectives

Customer purchasing behavior analyze karna

Gender aur age group ke according sales patterns identify karna

Top performing product categories identify karna

Different states aur occupations ke basis par sales trends dekhna

📊 Analysis Performed

Project me following analysis kiya gaya hai:

Data cleaning and preprocessing

Gender-wise sales analysis

Age group analysis

State-wise sales distribution

Occupation-based purchase analysis

Product category analysis

📈 Key Insights

Female customers ne male customers ke comparison me zyada purchasing ki.

26–35 age group ke customers ne highest sales generate ki.

Married women from IT, Healthcare, and Aviation sectors had higher purchasing power.

Uttar Pradesh, Maharashtra, aur Karnataka se highest orders aaye.
